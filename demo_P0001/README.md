# 시연 패키지 — P0001 (알바 대학생 · 투자 무경험)

전체 데이터셋(2,000명, 969MB)에서 **시연 11컷에 필요한 것만** 뽑은 880KB 패키지.
원본: `finmate3/data-gen/outputs/finmate_v3/` (2026-07-13 NaN 패치 반영본)
추출 스크립트: `../extract_p0001.py` (재실행 가능)

## 구성

```
demo_P0001/
├── demo_summary.json      ← 먼저 볼 것: 히어로 프로필·스탯·친구그룹·테이블 행수 요약
├── bundle/                ← P0001 원본 번들
│   ├── profile.json         21세 대학생, 용돈+알바 월 120만, 투자 X, 가입 2026-07-13
│   ├── ledger.csv           거래 450건 (2026-01-01 ~ 07-13)
│   └── api/                 마이데이터 응답 JSON 18개 (은행3·카드1·전금3 계정)
├── friends_profiles.json  ← 친구 12명(P0002~P0013) 프로필 요약
├── l3/                    ← 파생 테이블 (P0001 + 친구 12명으로 필터)
├── goal_templates.json    ← 목표 12종 (GOAL-SAVE-LUMP = "자금 모으기")

└── quiz_bank.json         ← 금융 퀴즈 25문항 (ETF O/X 3문항)
```

## 주의

- `profile.financial_goal`은 "등록금 완충"(여행 아님)이지만 **화면 비노출 내부 필드**이며, 메인퀘가 미설정이라 컷3에서 "유럽여행경비"를 새로 세우는 흐름과 충돌하지 않음.
- 친구들은 활발히 활동 중(퀘스트 794건·포인트 532건·레이드 127건)이고 히어로만 백지 상태 — 온보딩 시연에 최적.
- `l3/friendships`는 친구의 친구까지 포함(165행)하니 P0001 직접 친구만 쓸 거면 `friends_profiles.json`의 12명으로 필터.
